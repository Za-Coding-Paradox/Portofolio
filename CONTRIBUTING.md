# Contributing

This is a personal portfolio — feel free to fork it and adapt it for your own use.

## If you spot a bug

Open an issue describing:
1. What you expected to happen
2. What actually happened
3. Steps to reproduce

## If you want to suggest an improvement

Open an issue or submit a pull request. Keep changes focused and minimal.

## Code style

- No frameworks or build tools — keep it vanilla HTML/CSS/JS
- Use CSS custom properties for any new colours or tokens
- Keep JS readable — no minification in source
