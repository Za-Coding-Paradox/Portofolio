# Changelog

All notable changes to this project will be documented here.

## [1.0.0] — 2025

### Added
- Initial release
- Dark / Light / Terminal theme system
- `Ctrl+K` command palette for navigation
- Live GitHub dashboard (repos, stars, activity feed, contribution heatmap)
- Animated canvas starfield background
- Typewriter hero section
- Scroll-reveal animations
- Sections: Hero, About, Skills, Projects, Achievements, Research, GitHub, Footer
- Fully responsive layout (mobile, tablet, desktop)
- Separated CSS (`assets/css/style.css`) and JS (`assets/js/main.js`) from HTML
- Local profile image asset (`assets/images/profile.jpeg`)
